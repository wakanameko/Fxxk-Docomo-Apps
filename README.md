# Fxxk-Docomo-Apps
AndroidのDocomoのゴミアプリを消すことだけに集中したソフトです。
一応削除したらまずそうなアプリは削除しないようにしてますが、問題が発生する可能性があります。(発生したら＠wakanameko2のtwitterまでDMしてください)
